In a previous post (How to Install Kali Linux), we installed kali Linux with our bare hands, though it takes a whole day to set up and it is not easy. So, if you don't want to mess with settings, you can always install kali with the easy method, with which you will receive the default (preconfigured) settings. We will use an .ova (extension which is recognizable by VirtualBox) to install Kali:

TIP: The most common reason your Virtual Box is crashing/lagging/unable to start/unable to shut down is because of bottlenecks. Make sure to let it have more cores and ram if your GPU is good and less cores and ram if your GPU is crappy

Step 1: Configuring Windows Settings

First of all, you need to make sure that your machine supports virtualization. Otherwise you will not be able to boot up the virtual machine. Secondly, if your device supports virtualization you can continue and should have virtualization enabled from bios.

Step 2: Downloading & Installing the Software

To start you'll need VirtualBox, so download and install it:

Once you press on windows hosts the download should start. When it's finished open the exe and run the setup.

You will also need the guest addition (it is optional). This includes: Fullscreen, copy and paste between host and guest and much more! To download it, click here: VirtualBox Extension Pack. Open it and install it.

Next, we need the .ova file. Visit Get Kali and click Virtual Machines:

Here, select the VirtualBox versionas we are installing kali on VirtualBox, if you're installing on VMware, select VMware

and the download will start. After kali is finished downloading we need to install it.

Step 3: Setting Up Kali

In VirtualBox press on tools and then import:

Here, choose the location of the kali-linux-2022.2-virtualbox-amd64.ova. It will be on the default location of your browser's downloads (Most probably C:\Users\UserName\Downloads)

When you import the ova file, press next and here, we need to configure the CPU and RAM settings (You can change everything later). you can choose whatever you want as long as your pc can handle it I chose these settings: CPU 4 (Cores) | RAM 4096MB

You can also change the name and whatever you want, I am going to leave everything else as is. Once you have decided, press import and then wait, it could take from a minute to several minutes (depending on your pc):

To import the VirtualBox Extension Pack, choose your new machine, press Settings:

Select Storage:

Choose the optical drive button:

Click Add, browse to the location of the Addition Pack and click choose:

Click OK and you're good to go, press Start and when it asks you for username and password input root for both or kali for both (it depends on the version and I suggest you change it later). Open the terminal and write "sudo apt update", when its updated, type "sudo apt -y full-upgrade -y" to upgrade it.

So you have a fully functional Kali Linux Virtual Machine! If you have any errors, please leave a comment and I will try to answer as soon as I can.
