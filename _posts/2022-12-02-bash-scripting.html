Bash Scripting Notes


Bash Scripting Notes

File Operations

  • Opening a File in Edit Mode:
nano filename.sh
  • Running a Bash File:
    • bash filename.sh
  • Changing File Permissions:
    • chmod +x filename.sh
  • Running an Executable File:
    • ./filename.sh

    Console Output and Delay

    • Sending a Message to the Console:
    echo "hey"
  • Pausing Execution for a Specified Duration:
    • sleep 5

    Variable Operations

    • Creating a String Variable:
    variableName="this is text"
  • Utilizing a String Variable:
    • echo "this is a random text $variableName"
  • Reading Text Directly from the Console:
    • read variableName

    Command Line Arguments

    • Creating Arguments:
    arg1=\$1
arg2=\$2
  • Using Output of a Command as an Argument:
    • dateArgument=\$(date)
  • Creating System-Wide Variables:
    • export systemVariable="value"
  • Making a System Variable Permanent:
    • nano .bashrc
export systemVariable="value"

    Built-in Variables and Operations

    • Generating a Random Number:
    echo \$RANDOM
  • Performing Arithmetic Operations:
    • echo $((1+1))
  • Shell Information:
    • echo \$SHELL
  • User Information:
    • echo \$USER
  • Current Working Directory:
    • echo \$PWD
