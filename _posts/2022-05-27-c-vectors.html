Introduction to Vectors in C++

Vectors in C++ are dynamic arrays utilized to store data. Unlike static arrays, which store sequential data, vectors offer enhanced flexibility. Vectors can automatically resize based on the insertion or deletion of an element. This dynamic resizing feature distinguishes vectors from arrays, where only a predefined number of values can be stored under a single variable name. Data in vectors is typically inserted at the end.

Key Notes:

  • Note 1: Vector elements can be accessed similarly to array elements, using the syntax: vector[m].
  • Note 2: Directly assigning values to vector elements without using appropriate modifiers will not adjust the vector's size. For instance, vector[m] = n; is not recommended. Instead, it's essential to understand and utilize vector modifiers.

Vector Initialization:

To declare a vector, use the following syntax:

vector<object_type> variable_name;

Example:


#include <vector>
using namespace std;

int main() {
    vector<int> intVector;
    vector<string> stringVector;
}

Vector Iterators:

  • begin(): Returns an iterator pointing to the first element in the vector.
  • end(): Returns an iterator pointing to the theoretical element following the last element in the vector.

Vector Modifiers:

  • assign(n, m): Clears the vector and assigns the element m, n times.
  • push_back(m): Inserts the element m at the end of the vector, increasing its size by one.
  • pop_back(): Removes the last element from the vector, decreasing its size by one.
  • insert(m, k, n): Inserts the element n at the position m for k times. If k is omitted, n is inserted once.
  • erase(n, m): Removes elements from position n to position m. If m is omitted, only the element at position n is removed.
  • swap(): Exchanges the contents of the vector with another vector of the same type.
  • clear(): Removes all elements from the vector.

Examples of Vector Modifiers:


vector<int> exampleVector;

// Using assign():
exampleVector.assign(100, 10); // Fills the vector with 100 tens.
exampleVector.assign(3, 5);    // Clears the vector and adds three fives.

// Using push_back():
exampleVector.push_back(10);   // Adds a 10 to the end of the vector.

// Using pop_back():
exampleVector.pop_back();      // Removes the last element from the vector.

// Using insert():
exampleVector.insert(exampleVector.end(), 5);           // Inserts a 5 at the end.
exampleVector.insert(exampleVector.begin() + 1, 2, 3);  // Inserts two 3s at the second position.

// Using erase():
exampleVector.erase(exampleVector.begin() + 1);         // Removes the second element.
exampleVector.erase(exampleVector.begin(), exampleVector.end() - 1); // Removes all elements except the last one.

// Using swap():
vector<int> anotherVector;
exampleVector.swap(anotherVector); // Swaps the contents of exampleVector and anotherVector.

// Using clear():
exampleVector.clear(); // Removes all elements from exampleVector.
