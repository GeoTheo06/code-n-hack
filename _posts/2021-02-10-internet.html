Network Notes on IP Addresses

Public IP Address

A public IP address, also known as an external IP address, is a unique identifier assigned to every device that connects to the Internet. This protocol facilitates devices in locating each other on the global network for the exchange of information. End-users do not have control over the public IP address assigned to their device. The Internet Service Provider (ISP) assigns the public IP address when a device initiates an Internet connection. Public IP addresses can be classified into different types, namely: static, dynamic, or shared. These classifications dictate whether the IP address will change over time.

Static IP Address

A static IP address remains constant and is assigned exclusively to a single user, device, server, or website.

In the illustrated example, the Internet Service Provider (ISP) allocates distinct and immutable IP addresses to different routers. In such cases, the router is responsible for the Network Address Translation (NAT) process, making the ISP "transparent" when the router communicates with remote computers over the Internet.

For most end-users, a static IP address is not essential. However, it becomes crucial when continuous remote access to a device is required. Since the IP address remains constant, only one IP address needs to be remembered for repeated access.

Dynamic IP Address

A dynamic IP address may undergo periodic changes. These changes can occur when a connection is reset or through occasional updates by the ISP.

In the context of a dynamic IP address, the ISP assigns routers with private WAN IP addresses. These private addresses are translated to public IP addresses for external communications. Unlike static IP addresses, dynamic IP addresses are transient and subject to change.

While dynamic IP addresses pose challenges for remote access, solutions do exist. Utilizing Dynamic DNS services can simplify remote connectivity. These services update the IP address mapping to a hostname periodically, facilitating constant accessibility even if the IP address changes.

Private IP Address

A private IP address is restricted for use within a local network. Devices with private IP addresses cannot establish direct connections to the Internet. Additionally, devices outside the local network cannot initiate direct connections to a device bearing a private IP address.

The Internet Assigned Numbers Authority (IANA) has designated the following IP address ranges for private networks:

  • 10.0.0.0 - 10.255.255.255 (Total Addresses: 16,777,216)

  • 172.16.0.0 - 172.31.255.255 (Total Addresses: 1,048,576)

  • 192.168.0.0 - 192.168.255.255 (Total Addresses: 65,536)

IP Address Terminology

  • Static: Indicates an IP address that remains constant.

  • Dynamic: Indicates an IP address that may change periodically.

  • Public: Indicates an IP address that is globally accessible.

  • Private: Indicates an IP address that is restricted to a local network.

  • Shared: Indicates an IP address used by multiple users.

  • Dedicated: Indicates an IP address used exclusively by a single user.
