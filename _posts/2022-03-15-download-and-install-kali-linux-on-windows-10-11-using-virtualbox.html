In my opinion Linux is the most powerful OS. Though, there are so many distributions to choose from. My personal favorites are Ubuntu and Kali. As we are going to be working with Kali Linux in the next posts, I will first demonstrate the process of installing Kali Linux in a virtual machine.

Note that I am not responsible for any damage you cause on your Device. If you follow the instructions carefully you should be fine. You can also install kali using an easy method: Install Kali (the easy way).

TIP: The most common reason your Virtual Box is crashing/lagging/unable to start/unable to shut down is because of bottlenecks. Make sure to let it have more cores and ram if your GPU is good and less cores and ram if your GPU is crappy

Step 1: Configuring Windows Settings

First of all, you need to make sure that your machine supports virtualization. Otherwise you will not be able to boot up the virtual machine. Secondly, if your device supports virtualization you can continue and should have virtualization enabled from bios.

Step 2: installing the software

The first thing you need is VirtualBox (Downloads â€“ Oracle VM VirtualBox), the tool that we are going to use to install Kali. Since, you are installing Kali on windows, under "VirtualBoxÂ 'version' platform packages" as soon as you click on Windows Hosts, the Download will start.

Once it is downloaded, open it and go through the setup.

You will also need the guest addition (it is optional, though it will make your life 10x easier). This includes, Fullscreen, copy and paste between host and guest and much more! To download it, simply click on VirtualBox Extension Pack. Open it and install it.

Once you are done installing VirtualBox, you will need the iso image of kali Linux "Get Kali Linux". To download it, select "Live Boot",

and then select the latest version of kali, for me right now, it is 2022.1.

You can store the iso wherever you want, it doesn't matter because when the setup is done you won't need it.

Step 3: Setting Up Kali

On Virtual Box, press "new", and input the same settings as me:

Note: You should select 32-bit if your system is 32-bit.

click next and select the ram you want your virtual machine to have. I will select 4096 because i think its enough for Linux. Next, click create Virtual Disk now and select VDI. Click next, select dynamically allocated (preferably). Click next, and now you have to select the location in which the virtual machine will get installed and the amount of size you want your virtual machine to have. I set it to 50 GB though the minimum is 20 GB. Your Virtual Machine is almost ready!

Step 4: Configuring Kali's Settings

Settings Time! Select "Settings" (located next to "New"). In General Settings, under advanced, select bidirectional on both options:

In the system tab, under "Processor" choose your preferred number of CPUs you would like your kali to use (I have set it to 4). Note here, that the ram, processor and video card memory you set, are only used while running the virtual machine. As a newbie, I was worried they were used even if I didn't start the virtual machineðŸ˜…. Anyway, in the display tab go to video memory and set this setting to max. In the storage section, click on "Empty" and after you select the blue disk

click on "choose a disk file" and browse to the location of your iso.

Step 5: Installing Kali Linux

Now, it is finally time to boot up your new OS! On VirtualBox, select start and Kali will boot.

In case you get stuck on the boot screen, try installing an older version of VirtualBox. 5.2 is the version that worked for me.

Now, you will start installing Kali. To do that, select "Start Installer" from the menu that shows up when you start Kali.

Now before we start, I have devided the installation into 15 steps. Depending on what iso you downloaded, the steps will be a bit different (some settings on my iso will not exist on your installation so if they do not, just skip them, and if there are some steps in your installation that are not on mine, search them up).

1. Here, click your language, though I prefer that you select English so that you can go through the installer without problems. Click continue and

2. select your country. Click continue.

3. Select American English as keyboard layout, though, if you want you can also select any layout you want. Click continue and wait for it to install some drivers.

4. Next, choose a Hostname, (I will leave it "kali") it can be anything and click continue.

5. Now, choose your username (you will have to remember it so store it in a txt on windows) click continue.

6. choose a password and store as well on the txt or anywhere else you want. Click continue and

7. configure the clock based on where you live. Click continue, and now comes the most important part. From the screen that appears, select "guided - use entire disk":

click continue and

8. it will show you only one option as a disk, so select it and click continue. On the next window, choose the first option: "All files in one partition" though note here, that when playing with kali, some errors will occur, and when you search these errors in the internet, some people will give solutions that have files under a specific location like /tmp. So, after you learn how to navigate (it will take a month probably), I suggest you to run the installation again to select the third option from this window "seperate /home, /var, /tmp locations".

9. Click continue and on the next window, everything is configured properly so just click "Finish partitioning and write changes to disk" and continue.

10. On the next window, it will ask us whether we want to write the changes to the disk, and because we want, you will select "yes" and continue.

11. After it finishes writing the changes to the disk, it will ask you for a package manager and if you don't have one or don't even know what it is, leave it black and click continue.

12. Now, you need to select your desktop environment and tools. I will leave the desktop to xfce because this is what suits me. If you want to see how the desktops are, search for xfce vs GNOME etc. Anyway, just copy my settings, or if you are experienced, select what you want:

13. When you get to "install the GRUB loader on a hard disk" click yes and then continue.

14. Then select the second option: /dev/sda and click continue.

15. You are finally done! The installation is finishedand you are ready to see for the first (probably) time Kali Linux! Click continue and kali will restart, this time leading you to the kali desktop. Input your username and password from the txt and you are good to go!
